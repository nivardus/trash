# Trash

**Trash** is an online mini-sandbox that runs in your browser. This is the project's monorepo containing the client, server and art assets. It is in incredibly early development, barely a concept at this point, but the general ideas are:

- Users have persistent identity and can log in using an account they create
- The world is constructed of simple low-poly models and planes. Content creation can happen at runtime, in-world via a simple oekaki-style painting tool that allows users to paint onto any surface/model.

These is a ton of work to do:
